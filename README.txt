Taj Sales System Restore

This package restores a version similar to the earlier kg-based version and adds:
- Procurement prices saved until changed
- End-day accounting
- Fuel, km, consumption, hours, hourly rate
- Net profit calculation
- Dashboard with daily profit chart
- Backup and restore
- PWA offline support

Files:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js

Important:
Place your own icon.png in the same folder before uploading to GitHub Pages.
